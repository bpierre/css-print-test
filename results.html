@page rule

Testing the CSS2.1 @page rule with pseudo-selectors.
The only allowed CSS property in this context is margin and its suffixes.

  • Test 1 (show test) 
    @page{margin:6cm;}
  • Test 2 (show test) 
    @page{margin-top:6cm;}
  • Test 3 (show test) 
    @page{margin-bottom:6cm;}
  • Test 4 (show test) 
    @page{margin-left:6cm;}
  • Test 5 (show test) 
    @page{margin-right:6cm;}
  • Test 6 (show test) 
    @page:left{margin:6cm;}
  • Test 7 (show test) 
    @page:right{margin:6cm;}
  • Test 8 (show test) 
    @page:first{margin:6cm;}
